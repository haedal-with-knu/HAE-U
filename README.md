# 트랙 활동 내용
![로고](Unity.PNG)
 * 처음 유니티를 접해보는 트랙원들을 위해 2~3주의 기초 유니티 사용법 강좌가 있습니다.
 * 게임 개발의 기초를 배우신 트랙원들은 3~4명씩 팀을 이루어 각자만의 게임장르, 게임개발 목표를 정합니다.
 * 이렇게 제작한 게임으로 **경북대학교 코드페어**나 기타 공모전에 참가하는 것이 최종 목표입니다.
